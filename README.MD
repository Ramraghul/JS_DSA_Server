## API Endpoints

- **Check Project Version**
  - **Endpoint:** `GET /api/v1`
  - **Description:** Check the version of the JS_DSA_Server project.

- **Check Palindrome**
  - **Endpoint:** `POST /api/v1/js/palindrome`
  - **Request Body:**
    ```json
    {
        "value": "eye"
    }
    ```
  - **Description:** Check if the given string is a palindrome.
